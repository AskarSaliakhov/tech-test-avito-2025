# Система управления объявлениями для модерации

# Запуск проекта
1. Склонировать репозиторий 
- `git clone https://github.com/AskarSaliakhov/tech-test-avito-2025.git`
2. Перейти в `tech-int3-server`
- `cd tech-int3-server` # Перейти в бэкенд директорию   
- `npm run install` #Установка зависимостей
- `npm run dev` #Запуск сервера
3. Открыть новый терминал
4. Перейти в `client`
  - `cd client`# Перейти в фронтенд директорию
  - `npm run install`#Установка зависимостей
- `npm run dev`  #Запуск сервера

# Архитектура
Проект был проектирован с помощью структуры **Feature-Sliced Design**
- `app` - Структура с главными файлами (`App.tsx`, `main.tsx`, `index.css`)
- `pages` - Страницы приложения (Все объявления, одно объявление, Статистика)
- `wdigets` - Самостоятельные блоки приложений (Их объединяю в `pages`)
- `shared` - Переиспольземые компоненты. Сделал так, что компоненты принимали пропсы, чтобы в можно было их переиспользовать
